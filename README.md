# Chess
Terminal Chess Implementation
It has some rules such as:

 • Legal moves
 
 • Check and Checkmate
 
 • Stalemate
 
 • En passant
 
 • Castling
 
 • Fifty move rule

# Features

 • **user vs user Gameplay**: Classic chess matches with friends
 
 • **Random move generation**: Utilize Python's random library to generate moves randomly
